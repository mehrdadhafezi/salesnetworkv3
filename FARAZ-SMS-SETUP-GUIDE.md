# 📱 راهنمای کامل راه‌اندازی فراز اس‌ام‌اس (IPPanel)

## 🎯 مقدمه

این راهنما نحوه راه‌اندازی و استفاده از **فراز اس‌ام‌اس (IPPanel)** در پلاگین Sales Network را توضیح می‌دهد.

---

## 🔑 دریافت اطلاعات API

### مرحله 1: ورود به پنل
```
آدرس: https://app.farazsms.com
ورود با نام کاربری و رمز عبور
```

### مرحله 2: دریافت API Key
```
مسیر: تنظیمات > API Settings
کپی کردن: Api-Key
```

### مرحله 3: شماره خط
```
مسیر: خطوط من
کپی کردن: شماره خط اختصاصی (مثلاً: 5000xxx)
```

---

## 📋 تعریف الگو (Pattern) در پنل فراز

### مرحله 1: ایجاد الگوی جدید
```
مسیر: ابزارها > الگوهای پیامکی > ایجاد الگوی جدید
```

### مرحله 2: نوشتن متن الگو

**نکته مهم:** در فراز اس‌ام‌اس، متغیرها با `{{variable_name}}` تعریف می‌شوند.

#### مثال الگوی فاکتور:

```
{{customer_name}} گرامی، فاکتور {{invoice_code}} صادر شد.
پرداخت آنلاین: {{invoice_url}}
مبلغ قابل پرداخت: {{amount}} تومان
یا کارت به کارت به شماره: {{card_number}}
باشگاه مشتریان ماکسیمم
```

### مرحله 3: تعریف متغیرها

در قسمت تعریف متغیرها، باید دقیقاً همین نام‌ها را وارد کنید:

| نام متغیر | توضیح | مثال مقدار |
|-----------|-------|------------|
| `customer_name` | نام مشتری | مهرداد حافظی |
| `invoice_code` | کد فاکتور | INV-123456 |
| `invoice_url` | لینک پرداخت فاکتور | https://yoursite.com/invoice?code=... |
| `amount` | مبلغ فاکتور | 1,500,000 |
| `card_number` | شماره کارت برای کارت به کارت | 6037-9972-xxxx-xxxx |

### مرحله 4: ارسال برای تایید
```
دکمه "ارسال برای تایید" را بزنید
منتظر تایید از طرف فراز بمانید (معمولاً چند ساعت)
```

### مرحله 5: کپی کردن کد الگو
```
بعد از تایید، کد الگو نمایش داده می‌شود
مثال: invoice_pattern_v1
این کد را کپی کنید
```

## ⚙️ تنظیمات در وردپرس

### مسیر تنظیمات:
```
داشبورد وردپرس
→ Sales Network > تنظیمات
→ قسمت "پیامک"
```

### قدم 1: انتخاب سرویس
```
سرویس‌دهنده: فراز اس‌ام‌اس (IPPanel)
```

### قدم 2: وارد کردن اطلاعات API
```
API Key: [Api-Key از پنل فراز]
شماره فرستنده: [شماره خط اختصاصی]
```

### قدم 3: کد الگو
```
کد پترن فاکتور (فراز): [invoice_pattern_v1]
```

### قدم 4: متن پیامک (اختیاری)
```
اگر الگو خالی باشد، از این متن استفاده می‌شود:

مشتری گرامی {customer_name}، فاکتور شما با کد {invoice_code} صادر شد.
برای پرداخت: {invoice_url}
مبلغ: {amount} تومان
یا کارت به کارت به شماره: {card_number}
```

### قدم 5: شماره کارت
```
قسمت "کارت به کارت"
شماره کارت: 6037-9972-xxxx-xxxx
نام صاحب کارت: مهرداد حافظی
```

### قدم 6: ذخیره
```
دکمه "ذخیره تنظیمات" را بزنید
```

## 🔄 نحوه کار

### وقتی فروشنده فاکتور صادر می‌کنه:

```php
// 1. اطلاعات فاکتور
$customer_name = 'مهرداد حافظی';
$invoice_code  = 'INV-123456';
$invoice_url   = 'https://yoursite.com/invoice?code=INV-123456';
$amount        = '1,500,000';
$card_number   = '6037-9972-1234-5678';

// 2. ارسال به فراز با Pattern
send_faraz_pattern([
    'customer_name' => 'مهرداد حافظی',
    'invoice_code'  => 'INV-123456',
    'invoice_url'   => 'https://...',
    'amount'        => '1,500,000',
    'card_number'   => '6037-9972-1234-5678'
]);
```

### پیامک ارسال شده:
```
مهرداد حافظی گرامی، فاکتور INV-123456 صادر شد.
پرداخت آنلاین: https://yoursite.com/invoice?code=INV-123456
مبلغ قابل پرداخت: 1,500,000 تومان
یا کارت به کارت به شماره: 6037-9972-1234-5678
باشگاه مشتریان ماکسیمم
```

## 📊 مقایسه ارسال ساده vs الگو

### ارسال ساده (بدون Pattern):
```
✅ سریع‌تر در تنظیم
✅ بدون نیاز به تایید
❌ هزینه بیشتر
❌ احتمال فیلتر شدن بیشتر
```

### ارسال با الگو (Pattern):
```
✅ هزینه کمتر (50% تا 70% تخفیف)
✅ سرعت ارسال بالاتر
✅ احتمال فیلتر نشدن
✅ حرفه‌ای‌تر
❌ نیاز به تایید اولیه
```

## 🐛 عیب‌یابی

### مشکل 1: الگو کار نمی‌کنه
```
چک کنید:
☑️ کد الگو درست وارد شده؟
☑️ الگو تایید شده؟
☑️ نام متغیرها دقیقاً مطابق تعریف هستند؟
☑️ API Key صحیح است؟
```

### مشکل 2: Error 401 (Unauthorized)
```
دلیل: API Key اشتباه است
راه حل:
1. برو پنل فراز
2. تنظیمات > API Settings
3. Api-Key جدید بگیر
4. در وردپرس جایگزین کن
```

### مشکل 3: متغیر جایگزین نمیشه
```
چک کن:
❌ اشتباه: {{name}}
✅ درست: {{customer_name}}

❌ اشتباه: {{code}}
✅ درست: {{invoice_code}}
```

### مشکل 4: پیامک اصلاً ارسال نمیشه
```
چک کنید:
☑️ حساب شارژ هست؟
☑️ خط فعال است؟
☑️ IP سرور در whitelist پنل هست؟
☑️ فیلد موبایل مشتری پر شده؟
```

## 💰 هزینه‌ها

### ارسال ساده:
```
هر پیامک: ~200 ریال
```

### ارسال با الگو:
```
هر پیامک: ~100 ریال (50% تخفیف)
```

### مثال محاسبه:
```
1000 فاکتور در ماه:
- ارسال ساده: 1000 × 200 = 200,000 ریال
- ارسال الگو: 1000 × 100 = 100,000 ریال
صرفه‌جویی: 100,000 ریال در ماه
```
